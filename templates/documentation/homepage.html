Project Overview

We have designed a website to work with two datasets .
The website is composed by three parts which interact to analyze, convert the data in a usable format and present the results to the user. They are respectively "functions.py", "mediator.py" and "website.py".
Going a little bit more in detail, "functions.py" is the core of the analysis, it contains all the classes and methods needed in order to perform the operations on the datasets. "website.py" contains all the functions needed to interact with the user and present him the results. While "mediator.py" is the component which links the other two parts and allows them to communicate.
