Bug in the pagination in the webpage that let's you browse the datasets.

If you write in the form a negative number it will produce wrong values ad indexes and in the info.

To fix it go to the package folder and open __init__.py :

  • Windows: 'C:\ProgramData\Miniconda3\envs\SoftAppProject\Lib\site-packages\flask_paginate\__init__.py'

Then go to Pagination.__init__() and after 

             
         self.page = int(kwargs.get(self.page_parameter, 1))
at line 317 add

             
         if self.page < 1:
              self.page = 1
